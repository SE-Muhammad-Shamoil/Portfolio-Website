## 🚀 Key Features

- **Dynamic Data Fetching:** Directly integrates with the GitHub API to automatically fetch my live avatar and selectively render my most impressive curated repositories (e.g., Netflix Clone, Hill Cypher, Ibadify).
- **Modern Aesthetics:** Built entirely with pure standard **HTML5**, **Vanilla JavaScript**, and styled using **Tailwind CSS**.
- **Highly Interactive:**
  - Sleek Dark/Light mode theme toggling that automatically respects system preferences.
  - Glassmorphism UI elements and subtle staggered scroll-reveal animations.
  - Fully responsive layout engineered to perform seamlessly across desktop, tablet, and mobile breakpoints.
- **Academic Project Showcase:** A dedicated section highlighting my rigorous academic problem-solving skills in C/C++ (Decentralized File Explorer, Multithreaded Chat System).

## 🛠️ Technology Stack

- **Markup / Logic:** HTML5, Vanilla ES6+ JavaScript
- **Styling:** Tailwind CSS (via CDN for rapid styling), minor custom CSS for scrollbar and animation overrides.`
- **Icons:** Feather Icons API
- **Data Source:** GitHub REST API v3

## 💻 Running Locally

Since this project leverages a completely Vanilla stack without a complex build step, running it locally is extremely fast and simple.

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/SE-Muhammad-Shamoil/Portfolio-Website.git
    cd Portfolio-Website
    ```
2.  **Open in Browser:**
    You can simply double-click the `index.html` file to open it in your browser, or launch a local server for a better development experience:

    _Using Visual Studio Code:_
    Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension and click "Go Live".

    _Using Python:_

    ```bash
    python -m http.server 8000
    ```

    Then, navigate to `http://localhost:8000`.
